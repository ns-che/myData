ctrl+클릭 : 코드 경로 부분으로 이동 ex) src/style.css 를 클릭할 경우 해당 파일 수정으로 이동함
alt+클릭 : placeholder 클릭된 위치에 추가

ctrl+a 전체 선택
ctrl+f 단어 찾을때 활용
ctrl+h 단어 한번에 다른 단어로 수정할때 활용(페이지가 큰 경우)
ctrl+shift+l 단어 한번에 다른 단어로 수정할때 활용(페이지가 작은 경우)

shift+alt+f 선택된 부분 자동정렬
ctrl+alt+↕ placeholder 추가 => 화면이 뒤집어질경우 바탕화면 오른쪽 마우스->그래픽 옵션->바로가기키->사용안함
ctrl+x 한줄 잘라내기(복사됨)
alt + ↕ 현재 선택된 줄 (선택된 영역이 아닌 영역이 포함된 줄) 이동
alt+shift+↕ 현재 선택된 줄(선택된 영역이 아닌 영역이 포함된 줄) 복사
ctrl+shift+e 탐색기 쪽으로 포커스 이동
