1. HTML
1) 시맨틱 태그(Semantic Tag) => 코드를 읽는 사람들의 가독성을 위해 활용
관례
article : 게시물
section : 그 외 논리적 구분
div : 구분 없이 블럭요소 필요할때(의미적으로 관계가 없을때) - 줄바꿈 일어남, 사각형 박스
span:  구분 없이 블럭요소 필요할때(의미적으로 관계가 없을때) - 줄바꿈 X, 문장단위

2) class와 id 구분
class : 같은 속성을 지닌 태그
id : 같은 속성 안의 유일함을 구분하기 위한 태그
