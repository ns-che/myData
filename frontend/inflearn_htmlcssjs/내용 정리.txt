웹 페이지는 html + css + js(javascript)로 구성

ㅇ HTML 파일 작성
1. 먼저 head 부분(웹 브라우저 렌더링 전에 동작하는 부분)에 세팅이 필요하다.
1) 캐릭터 셋 세팅 = utf-8
2) html 요소들이 최신버전으로 렌더링 되도록 세팅
3) 반응형 웹 관련 세팅
4) 탭 이름 세팅
5) 아이콘, 폰트, 유저가 만든 스크립트(js)를 동작하기 위한 html 코드 세팅

2. body부분에는 스크립트를 가져오는 태그를 붙인다
=> <script src="Navbar.js"></script>

3. body 부분을 꾸민다
1) 네비게이션
header 태그 안에 놓는다

4. css, js 적용과 태그 제어를 위한 id, class를 설정한다


ㅇ CSS 파일 작성
1. 모든 속성에 적용
* {}
2. 태그에 적용
body{}
3. 해당 클래스 이름을 가진 태그에 적용
1) .클래스 이름{}
html에서 class="abc def"의 경우 .abc 하나만 해도 적용됨

2) 클래스 안의 클래스 = 두개의 클래스가 적용된 경우에만 적용됨
2-1) html에서 class="abc def"의 경우 .abc.def의 경우 적용됨
.클래스 이름.클래스 이름{}
2-2) html에서 class="abc" class="def 2개의 태그로 구성된다면 .abc > .def의 경우 적용됨
.클래스 이름 > .클래스 이름{}

3) 클래스 하위 요소들에 적용
.클래스 이름 *{}

.클래스 이름.클래스 이름{}
4. 자바스크립트가 만들어낼 곳의 스타일도 작성

4. 동적 요소 작성
1) 애니메이션 효과
@keyframes 이름{
	from{}
	to{}
}
@-webkit-keyframes 이름{
	from{}
	to{}
}

2) 특정 조건하에 발생
@media(min-width: 992px){ /*992px 이상일 경우만 해당 동작*/
	1,2,3에 쓰였던 문법
}
@media(min-width: 1200px){ /*1200px 이상일 경우만 해당 동작 => 992px 동작을 덮어씀*/ 
	1,2,3에 쓰였던 문법
}
